# TaskMatrix

**TaskMatrix** is a collaborative project management platform designed to streamline team workflows and enhance productivity. With a focus on simplicity and efficiency, TaskMatrix helps teams orchestrate their work with precision.

## Features

-   **Real-time Collaboration**: Seamlessly work together with team members on projects and tasks.
-   **Task Management**: Create, assign, and track tasks with intuitive drag-and-drop boards (Kanban) and list views.
-   **Instant Messaging**: Built-in chat functionality for real-time communication within projects and direct messages.
-   **Team Directory**: Manage team members, roles, and permissions easily.
-   **Secure Authentication**: Robust user authentication and profile management including avatar uploads.
-   **Visual Workflows**: Visualize project progress with dynamic boards and analytics.

## Tech Stack

**Frontend:**
-   React
-   Redux Toolkit
-   Tailwind CSS
-   Framer Motion
-   Vite

**Backend:**
-   Node.js
-   Express
-   MongoDB (Mongoose)
-   JWT Authentication
-   Socket.IO (optional real-time features)

## Getting Started

### Prerequisites

-   Node.js (v14+)
-   MongoDB Atlas URI

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/taskmatrix.git
    cd taskmatrix
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Run Locally:**
    ```bash
    npm run dev
    ```
